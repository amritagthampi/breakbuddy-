### Project Description
[Screen Time Break Alert is a smart browser extension that monitors active screen usage using keyboard and mouse activity. It enforces healthy work–break cycles by triggering guided exercise sessions before allowing users to resume work. The system promotes better eye health, posture, and productivity through structured digital well-being management.]

### The Problem statement
[Excessive screen time without structured breaks causes physical strain, reduced productivity, and long-term health issues. Existing reminder tools are easy to ignore and do not enforce meaningful break activities. There is a need for an intelligent screen-time management system that tracks active usage and ensures users perform guided physical exercises before resuming work.]

### The Solution
[Our solution is an interactive browser extension that monitors active screen usage and enforces structured break sessions. It detects keyboard and mouse activity to track real working time and triggers break alerts after user-defined intervals. During breaks, users must complete guided exercises such as breathing or stretching before resuming work. A limited emergency skip option is included for urgent situations. This ensures users take meaningful, health-focused breaks instead of ignoring reminders.]
